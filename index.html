前端入门

       首先欢迎各位小鲜肉们进入百步梯技术部，相信大家通过了前两轮的面试，已经对百步梯技术部的定位以及所要在部门学习的内容已经有了一定的了解。这篇文章主要是从前端的角度对web开发进行入门介绍。本人也很辣鸡，如果文章有不准确之处，还请及时指正。最后感谢大家点开了这篇文章。

什么是前端？

其实，大家每天都在接触前端。当你在手机或者浏览网页时所看到的其实就是网站前台，也就是我们说的前端。前端就是运行在PC端、移动端等浏览器上展现给用户浏览的网页。那前端工程师要做的事是什么呢？就是编写代码实现这些网页的内容和功能。前端开发需要特定的语言——HTML、CSS、JavaScript（最新的高级版本为HTML5、CSS3、SVG等），这是前端开发必备的三件套。

三件套介绍

HTML

简单来说，HTML的作用就是对页面内的元素进行定义。HTML全称指的是超文本标记语言 (Hyper Text Markup Language)。通过浏览器的识别渲染，这些HTML文件被“翻译”成了我们所见到的网页。我们要怎么在网页里找到这些文件呢？稍后我会为大家介绍。

CSS

HTML只是对页面内的元素进行定义，但是元素的默认样式和甲方的设计图差的有点远，这时候就需要一种工具对元素进行再加工，这个工具就是级联样式表（Cascading Style Sheet）简称“CSS”，通常又称为“风格样式表（Style Sheet）”。CSS的功能就是对进行网页风格设计，比如某个按钮的大小形状颜色、页面的背景和布局、点击某个元素之后元素的样式变化之类的。

JavaScript

在我的理解中，HTML和CSS其实算不上正式的编程语言，因为它们不是用来定义计算机程序的形式语言。

JavaScript（JS）是一种属于网络的高级脚本语言，已经被广泛用于Web应用开发，常用来为网页添加各式各样的动态功能，为用户提供更流畅美观的浏览效果。举个例子，当用户点击一个跳转页面的按钮时，JS就是用来告诉浏览器“用户按了这里，你要跳转啦！”并且完成跳转功能的角色。

开发的基本工具

开发者工具

一般情况下，Windows/Linux 中在 Chrome 内按下F12，Mac 用户按下command + option(alt) + i即可打开开发者工具。你可以大胆放心的在开发者工具里进行任何操作，这不会对网页造成任何影响。你可以选择打开随意一个网页，或者直接用我的博客链接（

Elements/元素


图中红色方框里的代码就是构成这个页面的HTML代码，可以点击开发者工具的左上角的图标审查某个元素。

当你点击这个元素的时候，右边的开发者工具会定位到元素对应的HTML代码和CSS代码（见图），这时候你可以对元素的样式进行修改，注意这个时候的修改是可以实时显示在页面上的，但是并不会对网页产生任何影响。比如对元素的字体颜色进行更改。

Console/控制台


Console面板在开发用来调试的时候比较多，对于某个功能出错的时候前端就会先看看控制台有没有报错，控制台面板中一般会打印一些报错信息，类似于日志，可以定位到哪个文件，帮助我们定位问题。当然也可以用来打印信息（比如百度隐藏的招聘广告）。
